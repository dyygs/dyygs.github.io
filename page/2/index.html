UITextField 学习小记

发表于 2016-10-12  |  2016-11-14   |   分类于 ios学习笔记   |     |   阅读次数

设置边框属性

可选属性：

  1. UITextBorderStyleNone,无边框
  2. UITextBorderStyleLine,有边框
  3. UITextBorderStyleBezel,有边框和阴影
  4. UITextBorderStyleRoundedRect圆角

设置键盘类型

可选属性：

  1. UIKeyboardTypeDefault, 当前键盘（默认）
  2. UIKeyboardTypeASCIICapable, 字母输入键
  3. UIKeyboardTypeNumbersAndPunctuation, 数字和符号
  4. UIKeyboardTypeURL, URL键盘
  5. UIKeyboardTypeNumberPad, 数字键盘
  6. UIKeyboardTypePhonePad, 电话号码输入键盘
  7. UIKeyboardTypeEmailAddress, 邮件地址输入键盘

###键盘风格

可选属性：

  1. UIKeyboardAppearanceDefault， 默认外观，浅灰色
  2. UIKeyboardAppearanceAlert， 深灰 石墨色

对齐方式

垂直对齐:

可选属性：

  1. UIControlContentVerticalAlignmentCenter 居中对齐
  2. UIControlContentVerticalAlignmentTop 顶部对齐，默认是顶部对齐
  3. UIControlContentVerticalAlignmentBottom 底部对齐
  4. UIControlContentVerticalAlignmentFill 完全填充

水平对齐:

可选属性：

  1. UITextAlignmentLeft，左对齐，默认是左对齐
  2. UITextAlignmentCenter，
  3. UITextAlignmentRight，右对齐

其他属性

  1. 设置自定义键盘 .inputView
  2. 键盘上方视图 .inputAccessoryView
  3. 左右视图 .leftView (UITextFieldViewModeAlways…)
  4. 清除按钮 .clearButtonMode
  5. 再次编辑是否清空 .clearsOnBeginEditing
  6. 设置返回键 .returnKeyType
  7. 设置自动纠正 .autocapitalizationType

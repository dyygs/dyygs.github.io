Requests库简介

发表于 2016-11-11  |  2016-11-14   |   分类于 python学习笔记   |     |   阅读次数

官方api

###译文：

本文涵盖了Requests库所有的接口，对于一些依赖的外部库，本文中提供了最重要的一部分说明，并提供了规范文档的链接

####主要接口

  1. requests.request(method, url, **kwargs) 构造并发送一个request请求
  2. requests.head(url, **kwargs)
  3. requests.get(url, params=None, **kwargs)
  4. requests.post(url, data=None, json=None, **kwargs)
  5. requests.put(url, data=None, **kwargs)
  6. requests.patch(url, data=None, **kwargs)
  7. requests.delete(url, **kwargs)

参数：

  • method –请求方式
  • url –请求地址
  • params –（可选）查询参数
  • data –(可选)请求body
  • json –()请求body中的json
  • headers –(可选)header头
  • cookies –(可选)cookies
  • files –(可选) 形式：{‘name:’, file-like-objects}，或者{‘name:’, file-tuple}，文件元组可以有3种形式{‘filename’, fileobj}, {‘filename’,fileobj,’content_type’}, {‘filename’, fileobj, ‘content_type’, custom_headers}
    • auth –(可选)http认证方式
  • timeout –(可选)链接超时
  • allow_redirects –(可选)重定向
  • verify –(可选)SSL验证，默认为true
  • stream –(可选)如果false，response会被立刻下载
  • cert –(可选)证书
    Return: request.Response

UITextField输入字符限制(中英文混编)

发表于 2016-10-14  |  2016-11-14   |   分类于 ios学习笔记   |     |   阅读次数
题记：

在ios的开发中，总会遇到要对输入的字符长度进行限制，目前网上大多提供的方法都是单纯的对 [textField.text length] 的简单判断，显然这种方法在面对中英文混合的时候并不适用，因为我们想要限制输入20个字符的时候，对应的中文字符应该是10个。

处理步骤：
  • 添加监听
  • 统计已输入字符的length(中文个数 * 2 + 英文字符个数)
  • 统计在字符长度限制下的maxLength
  • 对于超过长度的字符串进行截取
示例代码：
NSInteger lengthOfChiness = 0, lengthOfEn = 0;
   NSString *toBeString = textField.text;
   NSInteger maxLength = 0;
   for(int i=0; i< [toBeString length];i++){
       unichar a = [toBeString characterAtIndex:i];
       //判断是否是中文字符
       if( a > 0x4e00 && a < 0x9fff)
       {
           //统计中文个数
           lengthOfChiness += 1;
       } else {
           //统计英文个数
           lengthOfEn +=1;
       }
       if ((lengthOfEn + lengthOfChiness * 2) <= 20) {
           //长度限制下的最大字符个数
           maxLength = lengthOfEn + lengthOfChiness;
       } else {
           textField.text = [textField.text substringToIndex:maxLength];
           //自定义弹出窗口文字
           [[iToast makeText:@"超过字符限制"] show];
       }
   }
